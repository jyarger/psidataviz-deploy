# Security Policy

PsiDataViz fetches and parses data from user-supplied public URLs. If you discover a security issue —
for example a parser that can be made to crash, hang, or read unintended resources — please report it
privately rather than opening a public issue.

**Contact:** jyarger@proton.me

Please include steps to reproduce and, if possible, a minimal sample file or URL. We'll acknowledge your
report and keep you updated on the fix. Thank you for helping keep the project and its users safe.
